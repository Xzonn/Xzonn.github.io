## 第一章 水质及水处理基本方法
### 1.1 水污染来源与分类
- 水污染：水体因某种物质的介入，而导致其化学、物理、生物或者放射性等方面特性的改变，从而影响水的有效利用，危害人体健康或者破坏生态环境，造成水质恶化的现象。
- 水污染的来源：
  - 自然污染：特殊地质条件、动植物腐烂、降雨淋洗大气冲刷地面、自然灾害等。
  - 人为污染：生活污水、农田排水、工业废水、矿山排水、垃圾渗滤液、大气沉降等。
- 水污染的分类：
  - 化学性污染：无机污染物质、无机有毒物质、需氧污染物质、植物营养物质、油类污染物质、有机有毒物质。
  - 物理性污染：悬浮物质污染、热污染、放射性污染。
  - 生物性污染：病原微生物。

### 1.2 水质指标与水质标准
- 水质：水和其中所含的杂质共同表现出来的物理学、化学和生物学的综合特性。
- 水质指标：表示水中杂质的种类、成分和数量，是判断水质的具体衡量标准。
  - 物理性水质指标：
    - 感官物理性状指标：温度、色度、臭和味、浑浊度、透明度等。
    - 其他物理性水质指标：总固体、悬浮固体、溶解固体、可沉固体、电导率（电阻率）等。
  - 化学性水质指标：
    - 一般化学性水质指标：pH、碱度、硬度、各种阳离子、各种阴离子、总含盐量、一般有机物质（TOC）等。
    - 有毒化学性水质指标：各种重金属、氰化物、各种农药、多环芳烃（PAHs）等。
    - 氧平衡指标：溶解氧（DO）、化学需氧量（COD）、生化需氧量（BOD）、总需氧量（TOD）等。
    - 植物营养物质：氮（TN）、磷（TP）。
  - 生物学水质指标：细菌总数、总大肠菌群数、藻类、各种病原细菌、病毒等。
- 重要水质指标：
  - 浑浊度：水中的不溶解物质对光线透过时所产生的阻碍程度。测定方法：浊度计。
  - 色度：分为真色（除去悬浮物质后）、表色（溶解物质、胶体物质、悬浮物质共同引起）。测定方法：铂钴标准比色法。
  - 固体：一定温度（常 $$103 \sim \pu{105^\circ C}$$）下，将水样蒸发至干所残余的固体物质总量。
    - 按溶解性分为溶解固体和悬浮固体（SS）。
    - 按过滤性分为总可滤残渣和总不可滤残渣。
    - 按挥发性分为挥发性固体（$$\pu{660^\circ C}$$ 灼烧减重，约略代表有机物含量）和固定性固体（$$\pu{660^\circ C}$$ 灼烧残渣约略代表无机物含量）。
  - 比电导（电导率）：$$\pu{25^\circ C}$$ 时长 $$\pu{1 m}$$、横截面积 $$\pu{1 m2}$$ 水中的电导值。
    - 习惯上用电阻率衡量，相距 $$\pu{1 m}$$、横截面积各为 $$\pu{1 cm2}$$ 的两片平行板电极插入水中测定电阻值。电阻率是电导率的倒数。超纯水电阻率 $$\pu{18.2 M\Omega*cm}$$。
  - 总含盐量（总矿化度）：水中所含的各种溶解性矿物盐类的总量。总含盐量 = 溶解固体 + $$\ce{1/2HCO3-}$$ = 阳离子总量 + 阴离子总量。测定方法：离子色谱、原子吸收、电感耦合等离子体等。
  - 碱度：水接受质子的能力，是水中所有能与强酸发生中和作用的物质所接受的 $$\ce{H+}$$ 的物质的量总和。主要考虑 $$\ce{CO3^2-}$$、$$\ce{HCO3-}$$、$$\ce{OH-}$$。
    - 测定方法：中和滴定法，标准浓度盐酸滴定，分别以酚酞和甲基橙作指示剂计算滴定耗用的盐酸溶液量，得到酚酞碱度 $$P$$ 和甲基橙碱度（总碱度）$$T$$。

- - 化学需氧量（COD）和耗氧量（OC）：在一定严格的条件下，水中各种有机物质与外加的强氧化剂作用时所消耗的氧化剂量，可间接、相对地反映水中有机物浓度。
    - 根据所加强氧化剂种类分为化学需氧量（COD，加 $$\ce{K2Cr2O7}$$）和耗氧量（OC，加 $$\ce{KMnO4}$$）。
    - COD 反映了水中受还原性物质污染的程度，包含了无机性还原物质。测定方法对化学难分解有机物质不能被反映。
  - 生物化学需氧量（BOD）：在有氧条件下，水中可降解的有机物被好氧微生物作用而发生分解和无机化所需要的氧量。
    - 分为两阶段，第一阶段为碳氧化阶段，第二阶段为硝化阶段。
    - 有机物被微生物降解具有一级化学反应动力学的特性，$$\cfrac {\mathrm dL}{\mathrm dt} = -k_1'L$$，积分得到 $$\ln \cfrac {L_t}{L_{\mathrm a}} = -k_1't$$。
    - 换成以 $$10$$ 为底的对数，$$k_1 = 0.434 k_1'$$，则 $$\lg \cfrac {L_t}{L_{\mathrm a}} = -k_1t$$，或 $$L_t = L_{\mathrm a} \times 10^{-k_1t}$$。
    - 令 $$x_t$$ 为 $$t$$ 时间内降低的有机物浓度，则 $$x_t = L_{\mathrm a}\left(a - 10^{-k_1t}\right)$$。此为第一阶段和生化需氧量的反应动力学公式。
    - 在 $$10 \sim \pu{30^\circ C}$$ 时，$$k_{1(T)} = k_{1(20)}(1.047)^{T - 20}$$。
    - $$L_{\mathrm a(T)} = L_{\mathrm a(20)}(0.02T + 0.6)$$。
    - 常用测定方法：稀释与接种法。
  - COD、BOD<sub>5</sub>、OC 的关系：一般情况下 COD > BOD<sub>5</sub> > OC，若 BOD<sub>5</sub>/COD > 0.3 宜采用生化处理。
  - 总有机碳（TOC）和总需氧量（TOD）：
    - 总有机碳：以碳的含量表示水中有机物质含量的综合性指标。测定方法：燃烧氧化-非分散红外吸收法。
    - 总需氧量：水样中的有机物在 $$\pu{900^\circ C}$$ 高温下燃烧变成稳定的氧化物时所需的氧量。理论上 TOD = 2.67TOC。
  - 总氮（TN）和总磷（TP）：是表征试题含植物营养物质水平的水质指标。
    - 总磷构成：各种磷酸盐和有机磷化合物，也存在于腐殖质、水生生物中。
    - 总氮构成：污水中有机氮、氨氮含量高，地下水、地表水中硝态碳含量高。
    - 测定方法：分光光度法、离子色谱法。
  - 生物监测：利用受到污染的生物在生态、生理和生化指标等变化反映和度量环境污染程度。
- 水质标准：
  - 环境基准：环境污染物对特定对象不产生不良或有害影响的最大剂量或浓度。
  - 环境质量标准：权力机关为保障人体健康、保护生物资源和环境，根据人群和生态系统的综合要求，制定的各种环境参数允许水平的法规。
  - 重要的国家水质标准：饮用水水质标准、地表水环境质量标准、污水综合排放标准、城镇污水处理厂污染物排放标准等。

### 1.3 水处理的基本原则和方法
- 废水的成分与性质：
  - 生活污水：水质组成比较稳定，但浑浊、深色且具有恶臭，呈微碱性，一般不含有毒物质。长含有大量的细菌、病毒和寄生虫卵。
  - 工业废水：除冷却水等较清洁的生产废水外，都含有各种各样的污染物质，成分性质复杂，危害巨大。
  - 农业废水：牲畜废水总固体和有机物含量高，含有大量致病菌。农田径流污染来源广泛、随机形成、受季节影响、污染具有潜伏性。
  - 径流污水：农田径流含大量总固体、需氧有机物、总磷、总氮、病原体。城市径流含大量总固体、各类有机物、重金属。均具有季节性、冲击性、不确定性。
- 水处理的基本原则：效法自然，水体的自净作用。
  - 物理过程：稀释、扩散、挥发、沉淀等。
  - 物化过程：氧化、还原、吸附、凝聚、中和等。
  - 生物和生化过程：微生物分解、植物吸收等。
- 废水在水体中的稀释和扩散：
  - 推流：污染物质由于河流流速的推动，沿着水流前进的方向运行。$$Q_1 = v\rho$$。
  - 扩散：污染物因浓度差异而由高浓度向低浓度处迁移。$$Q_2 = -K\cfrac {\mathrm d\rho}{\mathrm dx}$$。
  - 水体混合：从废水排入口到完全混合前，只有一部分河水参与稀释。
    - 混合系数 $$a = \cfrac {Q_1}{Q}$$，其中 $$Q_1$$ 是参与混合稀释的河水流量，$$Q$$ 是河水总流量。
    - 稀释比 $$n = \cfrac {Q_1}{q}$$，其中 $$q$$ 是废水流量。
    - 某断面污染物的浓度 $$\rho = \cfrac {\rho_1q + \rho_2aQ}{q + aQ}$$。

- 水体的生化自净：
  - 氧垂曲线：有机污染物排入水体后，河流中溶解氧的变化，如图 1-1 所示。
    - 水中溶解氧的变化：$$\cfrac {\mathrm dX}{dt} = \cfrac {\mathrm dX_2}{dt} - \cfrac {\mathrm dX_1}{dt}$$，其中 $$X_1$$ 为耗氧量，$$X_2$$ 为复氧量。
    - 耗氧速率与有机物衰减相一致，$$\cfrac {\mathrm dX_1}{dt} = -k_1'L$$。
    - 复氧速率与亏氧量成正比，$$\cfrac {\mathrm dX_2}{dt} = k_2'D = k_2'(S - X)$$，其中 $$D$$ 为亏氧量，$$S$$ 为饱和溶解氧量。
    - 求微分得 $$\cfrac {\mathrm dD}{\mathrm dt} = -\cfrac {\mathrm dX}{dt} = k_1'L - k_2'D$$，解析解为 $$D_t = \cfrac {k_1'L_{\mathrm a}}{k_2' - k_1'}\left(\mathrm e^{-k_1't} - \mathrm e^{-k_2't}\right) + D_0\mathrm e^{-k_2't}$$。
    - 临界点 $$\cfrac {\mathrm dD_t}{\mathrm dt} = 0$$，解得 $$D_c = \cfrac {k_1}{k_2}L_0\times 10^{-k_1t_c}$$，$$t_c = \cfrac {1}{k_2 - k_1}\lg\cfrac {k_2}{k_1}\left[1 - \cfrac {D_0(k_2 - k_1)}{k_1L_0}\right]$$。
    - 多数情况下 $$k_{2(T)} = k_{2(20)}(1.016)^{T - 20}$$。

- 水体中生物物种和数量的变化：
  - 将全过程分为五个区：清洁区、降解区、强分解区、恢复区、清洁区。
- 给水处理的基本方法：
  - 水源为地面水时，原水 → 混凝 → 沉淀 → 过滤 → 消毒 → 饮用水。
  - 水源为地下水时，只需消毒处理。
  - 对于有机微污染水源：生物滤池、活性炭吸附、臭氧氧化消毒、微滤或超滤等。
- 废水处理的基本方法：源头节水、重复利用、回收有用物质、妥善处理。
- 水和废水处理的基本方法：
  - 分类：物理法、化学法、生物法。
  - 处理流程：
    - 一级处理：预处理，主要祛除水中较大的悬浮物质。
    - 二级处理：常规生化处理，去除水中呈溶解和胶体状态的有机物质。
    - 三级处理：深度处理，进一步去除水中的氮、磷、难降解有机物和溶解盐类等。

## 第二章 水的物理化学处理方法
### 2.1 水中粗大颗粒物质的去除
- 处理对象：颗粒直径 $$\pu{0.1 mm}$$ 以上的砂粒、卵石、砾石、树枝、厨余垃圾等。
- 去除技术：筛分／截流、重力沉降、离心分离。
- 筛分／截流：格栅、筛网和微滤机。
  - 格栅和筛网通常是水处理厂、污水处理厂的第一个单元，用于去除水中粗大物质，保护后续机械设备，防止堵塞。
  - 格栅由平行的金属栅条制成，大型污水处理厂宜采用机械清除格栅。
  - 去除水中纤维、纸浆、藻类等较小的杂物时，可选用不同孔径的筛网。
  - 微滤是一种截流细小悬浮物的精过滤装置。
- 重力沉降：沉砂池。
  - 一般设在泵站、沉淀池前，去除水中砂粒、煤渣等比重较大的无机颗粒杂质和少量较大、较重的有机杂质。
  - 沉砂池内颗粒物自由沉降。澄清流量 $$Q = \cfrac {h}{t}A = uA$$，其中 $$h$$ 为沉降距离，$$A$$ 为容器截面积，$$u$$ 为沉降速度。
  - 均匀球形颗粒在静水中的沉降速度：$$u = \cfrac {g}{18\mu}(\rho_{\mathrm s} - \rho)d^2$$。
  - 沉砂池三种型式：平流式、竖流式、曝气式。
    - 平流沉砂池：构造简单、截流无机颗粒效果好且稳定、排砂方便。
    - 曝气沉砂池：削减有机成分，克服沉渣中夹杂有机物容易腐败变臭的缺点，同时预曝气有利于后续生物处理。
- 离心分离：利用高速旋转的物体产生的离心力场以分离废水中的悬浮固体的处理方法。
  - 离心力 $$F_{\mathrm c} = (m - m_0)\cfrac {v^2}{r}$$，其中 $$m$$ 和 $$m_0$$ 分别时颗粒和水的质量。
  - 重力 $$F = (m - m_0)g$$，分离因素 $$a = \cfrac {F_{\mathrm c}}{G} \approx \cfrac {rn^2}{900}$$。

### 2.2 水中悬浮物质和胶体物质的去除
- 处理对象：胶体微粒（$$\pu{1 nm}\sim\pu{1 \mu m}$$）。
- 去除技术：沉淀、混凝、澄清、过滤、气浮。

#### 2.2.1 沉淀
- 沉淀理论：
  - 悬浮颗粒在水中的沉降根据浓度及特性分为四种基本类型：
    - 自由沉降：颗粒离散，形状、尺寸、质量、沉速不受干扰。例：沉砂池、初沉池初期。
    - 絮凝沉降：颗粒相互粘结，尺寸、质量、沉速增加。例：混凝沉淀、初沉池后期、二沉池初期。
    - 拥挤沉降（成层沉降）：颗粒间形成一个整体成层下降，在清水和浑水之间形成明显的界面。例：高浊度水的沉淀、混凝沉淀、二沉池后期。
    - 压缩沉降：上层颗粒重力作用将下层颗粒间的孔隙水挤压出界面，使颗粒群被压缩。例：沉淀池的污泥斗、污泥浓缩池。
  - 自由沉降：
    - 净作用力 $$F_{\mathrm n} = F_{\mathrm g} - F_{\mathrm b} = (\rho_{\mathrm s} - \rho)gV_{\mathrm s}$$，阻力 $$F_{\mathrm d} = \cfrac 12 C_{\mathrm D}d\rho A_{\mathrm s}u^2$$。
    - 达到平衡时 $$u = \sqrt{\cfrac {4(\rho_{\mathrm s} - \rho)gd}{3C_{\mathrm D}\rho}}$$，层流区 $$C_{\mathrm D} = \cfrac {24}{Re}$$，雷诺数 $$Re = \cfrac {\phi\rho ud}{\mu}$$。
    - 对于球形颗粒，$$u = \cfrac {(\rho_{\mathrm s} - \rho)gd}{18\mu}$$。
    - 沉淀试验：对有效水深 $$H$$ 和沉淀时间 $$t_0$$，当沉速为 $$u_0$$ 时恰好沉降去除。则沉速 $$u \ge u_0$$ 可全部去除，$$u < u_0$$ 的颗粒只有部分去除：$$\cfrac {h}{H} = \cfrac {u}{u_0}$$。总去除率 $$E = (1 - x_0) + \cfrac {1}{u_0}\int_0^{x_0} u\mathrm dx$$。
  - 絮凝沉降：
    - 絮凝沉降悬浮物去除率不仅取决于沉淀速度，而且与水深有关。
    - 目前没有适当的数学关系式描述，只能通过沉淀试验预测沉淀效果。
    - 可通过插值法绘制水深和时间的等效率曲线。
  - 拥挤沉降和压缩沉降：当浑液面以等速沉降至一定高度后，沉速逐渐减慢。从等速沉降转入降速沉降的一点称为临界点。临界点前为拥挤沉降，临界点后进入压缩沉降（污泥浓缩）。

    - 沉淀池容积为 $$V$$，表面积为 $$A$$，进水流量为 $$Q$$，则 $$u_0 = \cfrac {H}{t_0} = \cfrac {Qt_0}{At_0} = \cfrac {Q}{A}$$，令 $$q = \cfrac {Q}{A}$$，称之为表面负荷或过流率。
    - 实际沉淀池通常取 $$q = \left(\cfrac {1}{1.25} \sim \cfrac {1}{1.75}\right)u_0$$，$$t = (1.5 \sim 2.0)t_0$$。
